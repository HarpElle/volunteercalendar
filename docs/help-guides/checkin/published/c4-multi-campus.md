# C4 — `c4-multi-campus`

## Multi-campus check-in

**Audience:** Admin at a church running more than one campus.

**Outcome:** You understand how rooms, services, and passes behave per campus.

---

### The campus selector

When your church has more than one campus, a campus selector lets you choose which one you're working in. Rooms and services are set up per campus, so what you see follows the campus you've selected.

[SCREENSHOT: c4-campus-selector]

---

### Per-campus rooms and services

- **Rooms** are configured separately for each campus — a room at one campus doesn't appear at another
- **Service times** are also per campus, so each location can run its own schedule

[SCREENSHOT: c4-per-campus-rooms]

---

### Apple Wallet passes across campuses

The location-aware pass works at any campus you've set GPS coordinates for. When a family is near that campus, their pass surfaces with that campus's name. Make sure each campus has its coordinates set so the lock-screen prompt fires.
